---
title: DataGrid
description: A advanced data table component that supports sorting, selections, filters and pagination.
---

Data tables are used to organize lists of high density data.
It uses `react-table` to manage internal state.

<ComponentLinks pro={{ gumroad: true }} />

## Import

```ts
import { DataGrid } from '@saas-ui/pro'
```

## Usage

DataTable uses `@tanstack/react-table` internally and supports all TableOptions which allows you to extend the DataTable with pagination, column resizing, filtering and more, see the [TanStack table documentation](https://tanstack.com/table/v8/docs/api/core/table) for more information.

### Clickable rows

cells with a `href` property will render the cell value in an `a`.
Using the DataGrid `onRowClick` handler you can trigger a click event on the link whenever the row is clicked.

```jsx
function ListPage() {
  const [selections, setSelections] = React.useState([])

  const columns = React.useMemo(() => {
    return [
      {
        accessorKey: 'name',
        header: 'Name',
        width: '200px',
        meta: {
          href: ({ id }) => `#customers/${id}`,
        },
      },
      {
        accessorKey: 'email',
        header: 'Email',
      },
      {
        accessorKey: 'company',
        header: 'Company',
      },
      {
        accessorKey: 'country',
        header: 'Country',
      },
      {
        accessorKey: 'employees',
        header: 'Employees',
        isNumeric: true,
      },
      {
        id: 'action',
        disableSortBy: true,
        disableGlobaFilter: true,
        header: '',
        cell: () => (
          <>
            <Button size="xs">Edit</Button>
          </>
        ),
        width: '100px',
      },
    ]
  }, [])
  return (
    <Page
      title="Customers"
      height="400px"
      position="relative"
      overflow="hidden"
      sx={{
        '& tbody tr:hover': {
          cursor: 'pointer',
        },
      }}
    >
      <BulkActions
        selections={selections}
        actions={
          <>
            <Button colorScheme="white" variant="subtle">
              Delete
            </Button>
          </>
        }
      ></BulkActions>
      <DataGrid
        columns={columns}
        data={dataTable.data}
        isSortable
        isSelectable
        isHoverable
        onSelectedRowsChange={setSelections}
        onRowClick={(row, e) => {
          // Find the first A and trigger a click.
          const link = e.currentTarget.querySelector('td a')
          link && link.click()
        }}
      >
        <DataGridPagination />
      </DataGrid>
    </Page>
  )
}
```

```tsx
function ListPage() {

        header: 'Employees',

      },

        id: 'action',
        disableSortBy: true,
        disableGlobaFilter: true,
        header: '',
        cell: () => (
          <>
            <Button size="xs">Edit</Button>
          </>
        ),
        width: '100px',
      },
    ]
  }, [])
  return (
    <Page
      title="Customers"
      height="400px"
      position="relative"
      overflow="hidden"
      sx={{
        '& tbody tr:hover': {
          cursor: 'pointer',
        },

    >
      <BulkActions
        selections={selections}

              Delete
            </Button>
          </>
        }
      />
      <DataGrid
        columns={columns}
        data={dataTable.data}
        isSortable
        isSelectable
        isHoverable
        onSelectedRowsChange={setSelections}
        onRowClick={(row, e) => {
          // Find the first A and trigger a click.
          const link = e.currentTarget.querySelector('td a')
          link && link.click()
        }}
        manualSortBy
        onSortchange={setSort}
        pageCount={data.total}

          pageSize: 1,
        }}
      >
        <DataGridPagination onChange={({ pageIndex }) => setPage(pageIndex)} />
      </DataGrid>
    </Page>
  )
}
```

### Access internal state

You can access the react-table `TableInstance` by passing a ref to `DataTable`.
Check out the [react-table documentation](https://react-table-v7.tanstack.com/docs/api/useTable#instance-properties) for all properties.

```jsx
function TableState() {
  const tableRef = useRef(null)

  return (
    <Page
      title="Customers"
      height="400px"
      toolbar={
        <Toolbar>
          <ToolbarButton
            onClick={() => {
              tableRef.current.toggleAllRowsSelected(
                !tableRef.current.getIsAllRowsSelected()
              )
            }}
            label="Select all rows"
            variant="primary"
          />
        </Toolbar>
      }
    >
      <DataGrid
        instanceRef={tableRef}
        columns={dataTable.columns.concat()}
        data={dataTable.data.concat()}
        isSelectable
      />
    </Page>
  )
}
```

<Info>
  If you run into type errors, you need to add `react-table-config.d.ts` to your
  `src`. To use the default DataGrid config you can simply re-export the config
  from `@saas-ui/pro`.
</Info>

```ts
// Copy to: src/types/react-table-config.d.ts
export * from '@saas-ui/pro/src/data-grid/react-table-config.d'
```
