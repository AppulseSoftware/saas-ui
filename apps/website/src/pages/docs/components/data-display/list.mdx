A `StructuredList` can be used to list more complex types of data than the standard Chakra UI StructuredList component.

<ComponentLinks

  npm={{ package: '@saas-ui/list' }}

/>

## Import

- `StructuredList`: The wrapper component that handles default composition.
- `StructuredListHeader`: A title to indicate a group of items.
- `StructuredListItem`: The list item container.
- `StructuredListItemIcon`: The list item icon.
- `StructuredListItemButton`: Component used for clickable items.
- `StructuredListItemLabel`: Component that displays the item primary and secondary text.
- `StructuredListItemTertiary`: Extra data displayed next to the item label.
- `StructuredListItemAction`: The secondary item action.

```ts
import {
  StructuredList,
  StructuredListHeader,
  StructuredListItem,
  StructuredListItemIcon,
  StructuredListItemButton,
  StructuredListItemLabel,
  StructuredListItemTertiary,
  StructuredListItemAction,
} from '@saas-ui/react'
```

## Usage

### Basic StructuredList

```jsx
<StructuredList
  items={[
    {
      primary: 'Elliot Alderson',
      secondary: 'Hacker',
      tertiary: <Tag>admin</Tag>,
    },
    {
      primary: 'Martin Wallström',
      secondary: 'CEO',
      tertiary: <Tag>owner</Tag>,
    },
  ]}
/>
```

### Clickable with action

```jsx
<StructuredList
  items={[
    {
      href: '#',
      icon: <Avatar name="Elliot Alderson" size="sm" />,
      primary: 'Elliot Alderson',
      secondary: 'Hacker',
      tertiary: <Tag>admin</Tag>,
      action: <IconButton icon={<SettingsIcon />} />,
    },
    {
      href: '#',
      icon: <Avatar name="Martin Wallström" size="sm" />,
      primary: 'Martin Wallström',
      secondary: 'CEO',
      tertiary: <Tag>owner</Tag>,
      action: <IconButton icon={<SettingsIcon />} />,
    },
  ]}
/>
```

### Composed

```jsx
<Card width="100%" maxW="320px">
  <Box as="nav">
    <StructuredList>
      <StructuredListItem as="a" href="#home">
        <StructuredListItemIcon as={FiHome} />
        <StructuredListItemLabel primary="Home" />
      </StructuredListItem>
      <StructuredListItem as="a" href="#inbox">
        <StructuredListItemIcon as={FiInbox} />
        <StructuredListItemLabel primary="Inbox" />
        <StructuredListItemTertiary>
          <Badge borderRadius="full">20</Badge>
        </StructuredListItemTertiary>
      </StructuredListItem>
    </StructuredList>
  </Box>
  <Divider />
  <Box as="nav">
    <StructuredList>
      <StructuredListItem as="a" href="#spam">
        <StructuredListItemLabel>Spam</StructuredListItemLabel>
      </StructuredListItem>
      <StructuredListItem as="a" href="#trash">
        <StructuredListItemLabel>Trash</StructuredListItemLabel>
      </StructuredListItem>
    </StructuredList>
  </Box>
</Card>
```

### Nested

```jsx
function NestedNav() {
  const { isOpen, getToggleProps, getCollapseProps } = useCollapse()
  return (
    <Card width="100%" maxW="320px">
      <Box as="nav">
        <StructuredList>
          <StructuredListItem as="a" href="#home">
            <StructuredListItemIcon as={FiHome} />
            <StructuredListItemLabel primary="Home" />
          </StructuredListItem>
          <StructuredListItem as="a" href="#inbox" {...getToggleProps()}>
            <StructuredListItemIcon as={FiInbox} />
            <StructuredListItemLabel primary="Inbox" />
            <StructuredListItemTertiary>
              <Badge borderRadius="full">20</Badge>
              {isOpen ? <ChevronDownIcon /> : <ChevronRightIcon />}
            </StructuredListItemTertiary>
          </StructuredListItem>
          <Collapse {...getCollapseProps()}>
            <StructuredList>
              <StructuredListItem as="a" href="#spam">
                <StructuredListItemLabel ps={12}>Spam</StructuredListItemLabel>
              </StructuredListItem>
              <StructuredListItem as="a" href="#trash">
                <StructuredListItemLabel ps={12}>Trash</StructuredListItemLabel>
              </StructuredListItem>
            </StructuredList>
          </Collapse>
        </StructuredList>
      </Box>
    </Card>
  )
}
```
