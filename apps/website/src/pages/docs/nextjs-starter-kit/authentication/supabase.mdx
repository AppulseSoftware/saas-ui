Saas UI supports Supabase authentication out of the box. To use it, you need to configure the Supabase client and the Supabase auth service.

## Install

Install the Supabase Javascript client and the Supabase auth service.

```bash
cd apps/web && yarn add @supabase/supabase-js @saas-ui/supabase @acme/auth-supabase
```

## Configure

Edit `apps/web/.env.local` and add your public Supabase URL and KEY.

```bash
NEXT_PUBLIC_SUPABASE_URL="https://x.supabase.co"
NEXT_PUBLIC_SUPABASE_KEY="x"
```

Edit `apps/web/features/auth/auth-service` and create the supabase authentication service.

```js
import { createAuthService } from '@saas-ui/supabase'

import { createSupabaseBrowserClient } from '@acme/auth-supabase/client'

export const supabase = createSupabaseBrowserClient()

export const authService = createAuthService(supabase)
```

Now your frontend is configured to use Supabase authentication.

Try it out by running `yarn dev:web` and navigating to `http://localhost:3000`.
